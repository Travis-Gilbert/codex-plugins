# Route Conventions

> File conventions, segment config, and route structure rules.
>
> **Source files**: entries.ts, segment-config/, validate-app-paths.ts,
> app-page.ts template, app-route.ts template, verify-root-layout.ts,
> generate-interception-routes-rewrites.ts

## File Conventions

| File | Purpose |
|------|---------|
| `page.tsx` | Route UI (required for a route to exist) |
| `layout.tsx` | Shared wrapper for child routes |
| `loading.tsx` | Loading UI (Suspense boundary) |
| `error.tsx` | Error UI (error boundary) |
| `not-found.tsx` | Not found UI |
| `template.tsx` | Re-mounting layout (rare) |
| `default.tsx` | Parallel route fallback |
| `route.ts` | API endpoint (mutually exclusive with page) |
| `opengraph-image.tsx` | Dynamic OG image generation |
| `sitemap.ts` | Sitemap generation |
| `robots.ts` | Robots.txt generation |
